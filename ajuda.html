Ajuda!

+ DATAS: Todas as datas estão validadas quanto ao calendário.

+ IDADE: Será preenchida automaticamente ao ser informada a DATA DE NASCIMENTO do paciente. Caso o mesmo não se lembre, clicar no botão "O paciente não sabe especificar" para inserir a idade aproximada do paciente.

+ RESULTADO: De acordo com as respostas em "SINAIS E SINTOMAS", além de estar relacionado com o peso e a idade como foi pedido no formulário original, o botão devolve o número de pontos. As respostas: "POSSUI TB" ou "NÃO TB" não será mostrada devido a possibilidade de alterações na interpretação do resultado.

+ HORA: Deve ser preenchida no formato 24 horas, sendo o campo da ESQUERDA para HORAS e o da DIREITA para MINUTOS.

+ R$: Por enquanto, pode-se apenas incluir valores inteiros.

+ PORCENTAGEM: Os campos que envolvem PORCENTAGEM apenas poderão receber valores de 0 a 100.

+ "Outros": Será permitido apenas a entrada de dados nesses campos se os mesmos forem selecionados.

+ ENVIAR FORMULÁRIO: A CGI ativada pelo primeiro formulario (Criterio Inicial de Triagem) cria, se nao existe, um arquivo XML e adiciona os dados iniciais do paciente. Os outros 4 formulários apesar de estarem parcialmente validados, nao tem acao alguma, sendo prevista a criacao de outras CGIs para inclusao de dados a pacientes ja existentes (A CGI em vigor nao faz qualquer validacao de entrada de dados, portanto pacientes com mesmo Numero Geral podem ser incluidos no arquivo.).A CGI cria tb um arquivo em XML de apenas um paciente, que consta o ultimo paciente adicionado para visualizacao apos inclusao.

+ LIMPAR FORMULÁRIO: Apaga todos os campos preenchidos no formulário.
